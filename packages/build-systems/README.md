# Build Systems
Project with template to run a react projects using: rollup or webpack, supporting: react, sass and server with live reloading

# Installation
Run command (Only if you dont runned npx lerna bootstrap)
```(bash)
yarn
```


# Run
Both run in port: 8000

For webpack
```(bash)
yarn start:webpack
```
For rollup
```(bash)
yarn start:rollup
```
