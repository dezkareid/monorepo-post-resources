# Gherkin React

Testing using gherkin specifications

## Install dependencies

```bash
yarn
```
or
```bash
npm install
```

## Run tests

Run testing in components

```bash
yarn test
```
or
```bash
npm run test
```

Run testing in cypress

```bash
yarn start
```
and in another tab

```bash
yarn cypress:run
```

## Run lint

Run gherkin lint

```bash
yarn gherkin-lint
```
